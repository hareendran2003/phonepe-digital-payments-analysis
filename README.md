# 📱 PhonePe Digital Payments Analysis

## 📌 Overview

Built an end-to-end data analytics project using Python to analyze PhonePe's digital payment ecosystem across India. The project explores transaction patterns, user adoption, device preferences, and demographic factors to uncover actionable business insights and identify growth opportunities.

**Workflow:** Data Loading & Cleaning → Exploratory Data Analysis (EDA) → Advanced Analysis → Data Visualization → Business Insights & Recommendations

---

## 🎯 Business Problem Statement

The rapid growth of digital payments in India has created the need to understand transaction behavior, user adoption, and regional differences in digital payment usage.

This project aims to analyze PhonePe transaction and demographic datasets to answer key business questions such as:

* Which states contribute the highest transaction volumes and values?
* How has digital payment adoption grown over time?
* Is there a relationship between population and transaction activity?
* Which device brands dominate different regions?
* Which states and districts show the highest user penetration?
* How can these insights support market expansion and strategic decision-making?

The objective is to generate actionable insights that can help digital payment companies improve customer acquisition, target underserved markets, and enhance user engagement.

## 📁 Dataset

The project uses multiple datasets containing transaction, user, device, and demographic information stored in a single Excel workbook with separate sheets.

| Sheet Name             | Description                                     |
| ---------------------- | ----------------------------------------------- |
| State_Txn_and_Users    | State-level transaction and user data           |
| State_TxnSplit         | Transaction type distribution by state          |
| State_DeviceData       | Device brand usage across states                |
| District_Txn_and_Users | District-level transaction and user data        |
| District_Demographics  | District population and demographic information |

## 🔍 Project Workflow

### 1️⃣ Data Loading & Preparation

* Loaded and explored multiple datasets.
* Examined dataset structure and summary statistics.

## 📈 Key Insights

* Digital payment transactions have grown significantly across India over time.
* Transaction value and transaction count show a strong positive relationship.
* User penetration varies considerably across states.
* Population and transaction activity exhibit a moderate positive correlation.
* Xiaomi and Samsung dominate device usage across many states.
* Several states and districts show significant growth potential despite lower current adoption.

---

## 💡 Business Recommendations

* Focus expansion efforts on states with low user penetration but high population potential.
* Develop targeted marketing campaigns for underserved regions.
* Partner with leading smartphone brands to improve customer acquisition.
* Create region-specific strategies based on transaction behavior and user adoption.
* Increase financial inclusion initiatives in low-adoption districts.
